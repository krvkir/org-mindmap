#+TITLE: org-mindmap

Editable mindmaps for org-mode. Just as simple as that.

#+begin_mindmap :layout left
┬─ Письмо ── помощник для ┬─ мышления
│                         ├─ чтения
│                         ├─ изучения
│                         ├─ понимания
│                         ╰─ генерации идей
├─ Мышление ── происходит на бумаге
├─ Правила ┬─ преврати в черновик
│          ├─ отредактируй
│          ╰─ держать ручку наготове
╰─ Процесс ── собери заметки ┬─ мимолётные :: напоминания о мыслях ┬─ помести в одно место
                             │                                     ╰─ обработай позже
                             ├─ о литературе ┬─ когда читаете
                             │               ├─ кратко
                             │               ├─ избирательно ── для своих тем
                             │               ├─ своими словами
                             │               ├─ с библиографическими данными
                             │               ╰─ в картотеку
                             ╰─ постоянные ┬─ просмотри заметки ┬─ мимолётные
                                           │                    ├─ о литературе
                                           │                    ╰─ раз в день
                                           ├─ подумай ── как соотносятся с ┬─ исследованиями
                                           │                               ├─ размышлениями
                                           │                               ╰─ интересами
                                           ├─ одну для каждой идеи
                                           ├─ как для другого ┬─ полные предложения
                                           │                  ├─ источники
                                           │                  ├─ ссылки
                                           │                  ├─ точно
                                           │                  ├─ ясно
                                           │                  ╰─ кратко
                                           ├─ выбрось ── мимолётные
                                           ├─ добавь ┬─ позади заметки, к которой относится напрямую
                                           │         ╰─ ссылки
                                           ├─ убедись ── что сможешь найти ┬─ в указателе
                                           │                               ╰─ в точке входа
                                           ╰─ разрабатывай темы ┬─ снизу вверх
                                                                ├─ посмотри ┬─ что есть
                                                                │           ╰─ какие вопросы возникают
                                                                ├─ читай, чтобы ┬─ аргументы ┬─ поставить под сомнение
                                                                │               │            ╰─ укрепить
                                                                │               ╰─ доводы ┬─ расширить
                                                                │                         ╰─ изменить
                                                                ├─ делай больше заметок
                                                                ├─ развивай идеи ── смотри, куда приведут
                                                                ├─ следуй за интересом
                                                                ╰─ выбирай путь, который ── обещает больше идей
#+end_mindmap
